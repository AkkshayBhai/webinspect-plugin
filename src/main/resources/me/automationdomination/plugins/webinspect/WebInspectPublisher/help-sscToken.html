Create an "AnalysisUploadToken" with your fortifyclient command-line utility, for authorizing access to upload your WebInspect .fpr scan file to your Software Security Center (SSC) deployment. Usage of how you create your "AnalysisUploadToken" is as follows:

fortifyclient -url http://localhost:8080/ssc token -gettoken AnalysisUploadToken -daysToLive 90 -user admin
